# MermaidFolderTree

> ⚠️ 100% vibe-coded

Visualizes a folder's structure as a [Mermaid](https://mermaid.js.org/) markdown file.

## Scripts

- **`treemap_folders.py`** — generates a `treemap-beta` diagram of the folder structure.
- **`mindmap_folder.py`** — generates a `mindmap` diagram of the folder structure.

## Usage

```bash
python treemap_folders.py
# Enter the path to the folder when prompted.
# Outputs: <folder>_treemap.md

python mindmap_folder.py
# Enter the path to the folder when prompted.
# Outputs: <folder>_mindmap.md
```
